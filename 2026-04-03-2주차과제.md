# VM1 - dns.claude.local (DNS Server)

IP : `172.16.11.XXX`

---

## 1️⃣ 호스트네임 변경

```bash
hostnamectl set-hostname dns.claude.local
exec bash
```

---

## 2️⃣ DNS 서버 설치

```bash
dnf install -y bind bind-utils
vi /etc/named.conf
```

### named.conf 수정

```
listen-on port 53 { any; };
allow-query     { any; };
recursion no;
```

맨 아래 추가

```
zone "claude.local" IN {
    type master;
    file "claude.local.zone";
};
```

---

## 3️⃣ Zone 파일 생성

```bash
vi /var/named/claude.local.zone
```

```
$TTL 86400
@       IN      SOA     ns.claude.local. root.claude.local. (
                        20260403
                        24H
                        15M
                        48H
                        86400 );

        IN      NS      ns.claude.local.

ns      IN      A       172.16.11.XXX
web     IN      A       172.16.11.XXX
ftp     IN      A       172.16.11.XXX
```

---

## 4️⃣ 설정 점검 및 서비스 시작

```bash
named-checkconf /etc/named.conf
named-checkzone claude.local /var/named/claude.local.zone

systemctl start named
systemctl enable named
```

---

## 5️⃣ 방화벽 설정

```bash
firewall-cmd --permanent --add-service=dns
firewall-cmd --reload
```

---

## 6️⃣ 네트워크 고정 IP 설정

```bash
nmcli con mod enp0s3 ipv4.method manual \
ipv4.addresses 172.16.11.XXX/24 \
ipv4.gateway 172.16.11.X \
ipv4.dns "172.16.11.XXX 8.8.8.8"

nmcli con up enp0s3
```

```bash
hostnamectl set-hostname web.claude.local
exec bash
```

---

## 2️⃣ 네트워크 고정

```bash
nmcli con mod enp0s3 ipv4.method manual \

ipv4.dns "172.16.11.XXX 8.8.8.8"
```

---

## 3️⃣ Apache + HTTPS 설치

```bash
dnf install -y httpd mod_ssl

systemctl start httpd
systemctl enable httpd
```

### 방화벽

```bash
firewall-cmd --permanent --add-service=http
firewall-cmd --permanent --add-service=https
firewall-cmd --reload
```

---

## 4️⃣ SSL 인증서 생성

```bash
mkdir /etc/httpd/conf/ssl

openssl req -newkey rsa:2048 -nodes \
-keyout /etc/httpd/conf/ssl/claude.key \
-x509 -days 365 \
-out /etc/httpd/conf/ssl/claude.crt
```

CN 입력

```
web.claude.local
```

---

## 5️⃣ ssl.conf 수정

```bash
vi /etc/httpd/conf.d/ssl.conf
```

```
SSLCertificateFile /etc/httpd/conf/ssl/claude.crt
SSLCertificateKeyFile /etc/httpd/conf/ssl/claude.key
```

확인 후 재시작

```bash
httpd -t
systemctl restart httpd
```

---

## 6️⃣ 웹페이지 생성

```bash
vi /var/www/html/index.html
```

# VM3 - ftp.claude.local (FTP Server)

IP : `172.16.11.XXX`

---

## 1️⃣ 호스트네임 변경

```bash
hostnamectl set-hostname ftp.claude.local
exec bash
```

---

## 2️⃣ 네트워크 설정

```bash
nmcli con mod enp0s3 ipv4.method manual \
ipv4.addresses 172.16.11.XXX/24 \
ipv4.gateway 172.16.11.X \
ipv4.dns "172.16.11.XXX 8.8.8.8"

nmcli con up enp0s3
```

---

## 3️⃣ vsftpd 설치

```bash
dnf install -y vsftpd

systemctl start vsftpd
systemctl enable vsftpd
```

---

## 4️⃣ 사용자 계정 생성

```bash
useradd claude
passwd claude

mkdir -p /home/claude/uploads

chown claude:claude /home/claude/uploads
chmod 755 /home/claude
```

---

## 5️⃣ SSL 인증서 생성

```bash
openssl req -newkey rsa:2048 -nodes \
-keyout /etc/vsftpd/vsftpd.pem \
-x509 -days 365 \
-out /etc/vsftpd/vsftpd.pem
```

CN 입력

```
ftp.claude.local
```

---

## 6️⃣ vsftpd 설정

```bash
vi /etc/vsftpd/vsftpd.conf
```

맨 아래 추가

```
ssl_enable=YES
rsa_cert_file=/etc/vsftpd/vsftpd.pem
rsa_private_key_file=/etc/vsftpd/vsftpd.pem

force_local_logins_ssl=YES
force_local_data_ssl=YES

ssl_tlsv1=YES
allow_anon_ssl=NO

pasv_enable=YES
pasv_min_port=50000
pasv_max_port=55000
```

---

## 7️⃣ 방화벽 설정

```bash
firewall-cmd --permanent --add-service=ftp
firewall-cmd --permanent --add-service=ssh
firewall-cmd --permanent --add-port=50000-55000/tcp

firewall-cmd --reload

systemctl restart vsftpd
```
